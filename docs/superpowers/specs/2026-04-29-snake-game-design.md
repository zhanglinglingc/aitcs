# 贪吃蛇 - 宇宙星空版 设计规范

## 概述

一款视觉华丽的单文件贪吃蛇游戏，主题为「宇宙星空」。玩家在浩瀚星空背景中操控发光蛇身，体验四种不同游戏模式。

## 视觉风格

### 主题：宇宙星空
- **背景**: 深度太空渐变 (#050510 → #0a0a20)，多层视差星星
- **星云**: 半透明紫色/蓝色云雾，慢速漂移
- **蛇身**: 青蓝色渐变 (#00c8ff → #0080ff)，外发光效果
- **蛇头**: 更亮的青色，带眼睛细节
- **食物**: 脉动紫色能量球 (#cc44ff)，带光晕
- **粒子拖尾**: 蛇身后留下渐渐消散的彗星粒子
- **得分动画**: 流星飞入计分板

### 字体
- 主标题: Orbitron (Google Fonts)
- 分数/UI: Orbitron

### 特效
- 星星闪烁动画
- 星云漂移层
- 蛇身粒子拖尾
- 食物脉动光晕
- 吃食物粒子爆炸
- 游戏结束屏幕震动

## 音效（Web Audio API 程序化生成）

| 事件 | 音效 |
|------|------|
| 背景 | 空灵电子氛围音 |
| 吃食物 | 能量吸收音（上升音调） |
| 得分 | 流星音效 |
| 游戏结束 | 低沉爆炸音 |
| 模式切换 | 电子点击音 |

## 游戏模式

### 1. 经典模式
- 无尽模式
- 碰撞墙壁或自身即结束
- 吃食物增长蛇身

### 2. 穿墙模式
- 经典规则 + 可穿越边界
- 从屏幕一侧穿出，另一侧进入

### 3. 关卡挑战模式
- 每关有目标分数
- 达到目标进入下一关
- 随关卡增加：速度提升、地图出现障碍物
- 障碍物为静态陨石，碰撞即结束

### 4. 道具模式
- 无尽模式
- 随机道具掉落（每隔 5-10 秒）
- 道具类型：
  - ⚡ 加速（蛇身变短2格，速度+50%，持续5秒）
  - 🛡️ 护盾（免疫一次碰撞）
  - ✨ 黄金食物（+50分）
  - 🔥 冰冻（速度-30%，持续5秒）

## 操作

- **方向键**: 控制蛇移动方向
- **Enter**: 确认选择 / 重新开始游戏
- **Escape**: 暂停 / 返回模式选择

## 界面流程

```
[开机画面] → [模式选择] → [游戏] → [结束画面] → [模式选择]
```

### 模式选择界面
- 4 张模式卡片，2x2 布局
- 方向键移动选择高亮
- Enter 确认进入
- 悬浮卡片有发光高亮效果

### 游戏界面
- 顶部 HUD: 当前分数、最高分、当前模式、关卡(关卡模式)
- 中央: 游戏画布 (Canvas)
- 底部: 操作提示

### 结束画面
- 半透明遮罩
- 最终得分（大字）
- 最高分记录
- "按 Enter 重新开始" 提示
- 3 秒内按其他键返回模式选择

## 技术实现

- **单 HTML 文件**: 所有 CSS/JS 内联，无外部依赖
- **Canvas 2D**: 游戏渲染
- **requestAnimationFrame**: 游戏循环
- **Web Audio API**: 程序化音效生成
- **localStorage**: 存储最高分
- **Google Fonts CDN**: Orbitron 字体
